# Inventory Management System

A modern and responsive **Inventory Management System** built using **React.js**, **Redux Toolkit**, **Firebase Realtime Database**, and **CSS3**.

This project allows users to manage products with full CRUD operations:

- Add Product
- View Product List
- Update Product Details
- Delete Product
- Realtime Firebase Database Storage
- Clean & Professional UI Design

---

## Live Features

### Product Management

- Add new products with:
  - Product Name
  - Category
  - Quantity
  - Price

### Product Table

- View all products in table format
- Edit any product instantly
- Delete products permanently

### Database

All data is stored in **Firebase Realtime Database**
